# MyWeather APP

W zakładce Rejestracja znajdziesz formularz rejestracji, który sprawdza poprawność wpisanych danych i informuje o ewentualnych błędach.

W zakładce Pogoda znajdziesz aplikację MyWeather. Jest to aplikacja, która pozwala szybko sprawdzić pogodę w dowolnym mieście na świecie. 

Wystarczy wpisać nazwę miejscowości, by w kilka sekund zobaczyć aktualną temperaturę, wiatr, wilgotność i wiele innych danych. Aplikacja pokazuje też prognozę na najbliższe 7 dni, a tło strony zmienia kolor w zależności od pogody — inaczej wygląda przy słońcu, deszczu czy burzy. 

Można ją zainstalować na telefonie lub komputerze jak zwykłą aplikację i korzystać z niej bez otwierania przeglądarki. 

Link do aplikacji: https://obionekenobi712.github.io/MyWeather/
